# Design Patterns

A comprehensive collection of design patterns organized by category.

## Categories

- **Behavioral Patterns** - Patterns focused on object collaboration and responsibility distribution
  - Strategy

- **Creational Patterns** - Patterns for object creation mechanisms
  - Factory

- **Structural Patterns** - Patterns for composing objects into larger structures
  - Adapter
  - Decorator

## Overview

This repository contains implementations and examples of various design patterns.

Each pattern directory contains code examples and documentation demonstrating the pattern's structure and usage.
